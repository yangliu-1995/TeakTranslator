How to Use Debug Mode

  1. Open Chrome DevTools (F12 or Cmd+Option+I)
  2. Switch to Console tab to view detailed logs
  3. Switch to Sources tab to view source code
  4. Find source code under webpack:// in the left file tree
  5. Click "Start Page Translation" to begin testing
