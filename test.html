Welcome to Teak Translator Test Page

Introduction

This is a test page for Teak Translator. The translator will automatically detect the environment and use the appropriate translation API.

Features

  • Automatic DOM traversal and translation
  • Support for dynamic content (Mutation Observer)
  • Visibility detection for performance optimization
  • Support for iframes and shadow DOM
  • RTL language support
  • Translation cache for better performance

How to Test

  1. Click Start Page Translation to translate the entire page
  2. Watch the console output for translation progress
  3. Click Stop Translation to restore the original content
  4. Select some text and open Chrome DevTools console
  5. Use the command provided in TEST.md for selection translation
